게시판은 Chrome 및 Microsoft Edge 최신 버전을 지원합니다.

지원하지 않는 브라우저에서는 게시판이 정상적으로 동작하지 않거나 일부 기능 사용에 제약이 있을 수 있으니 양해 부탁드립니다.

게시글 작성자 외에도 관리자 및 게시판 관리자는 관리 목적상 구성원이 작성한 글을 삭제할 수 있습니다.

게시글 작성자, 관리자 및 게시판 관리자를 제외한 사용자는 작성된 글을 삭제할 수 없습니다.

회원가입을 완료한 경우 : 트립스타 홈페이지 > 상단에 로그인(로그인 페이지 바로가기) > ID 입력 > 비밀번호 입력

회원가입을 완료하지 않은 경우 : 트립스타 홈페이지 > 상단에 회원가입(회원가입 페이지 바로가기) > 사용할 ID 입력 > 비밀번호 입력 > 비밀번호 확인 > 가입 > 서비스 이용

탑승객 영문명은 여권과 동일해야 합니다.
[예약 확인]에서 변경 신청이 가능합니다.

출국일/귀국일 변경은 기존 예약을 취소하고 다시 예약 하시는 것을 권장합니다.

여행/티켓 상품은 현금영수증을 오프라인에서 발급이 가능합니다.
다만, 판매자가 해외 사업자인 경우, 국내 소득세법이 아닌 해외 법이 적용되기 때문에 현금영수증 발행을 하지 않습니다.

상품에 따라 티켓 취소/환불규정이 다를 수 있습니다.

구매 확인 문자가 오지 않는 경우 구매자의 휴대폰 정보가 정확한 지 확인해 주십시오.

사용 당일이나 이미 사용처리된 경우 취소되지 않을 수 있습니다.
